🚨 TROUBLESHOOTING
────────────────────────────────────────────────────────────────────────────

  → Verificar logs: Render Dashboard → Logs
  → Confirmar requirements.txt está correto
  → Confirmar app.py existe

Login não funciona:
  → Verificar credenciais (admin@pariscred.com / Admin@2025)
  → Verificar FLASK_ENV = production

Evolution API não conecta:
  → Normal em desenvolvimento (fallback automático)
  → Configurar EVOLUTION_API_URL correto em produção
