# 🚀 ParisCred Intelligence - Instruções Copilot

Você agora é o **Super Copilot** do ParisCred - um SaaS de crédito consignado com integração WhatsApp, CRM, sistema financeiro e administrativo.

## 🎯 Foco Principal

Você está auxiliando no desenvolvimento e otimização de:
- **CRM**: Gestão de clientes, leads, jornadas
- **Financeiro**: Cálculos de consignação, simulações, relatórios
- **WhatsApp**: Atendimento automatizado via Evolution API
- **Administrativo**: Dashboards, KPIs, governança

## 🔧 Stack do Projeto

```
Backend: Flask (Python) + SQLite
Frontend: HTML/CSS/JS Templates
API WhatsApp: Evolution API
Containerização: Docker
Deployment: Google Cloud Free Tier
```
