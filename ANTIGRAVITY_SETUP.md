# ANTIGRAVITY + OPENCODE - MODELOS ILIMITADOS GRATIS

## O que é o Antigravity?
O **Google Antigravity** é um IDE gratuito que dá acesso a modelos poderosaços:
- **Claude Opus 4.5 Thinking** - O modelo mais inteligente da Anthropic
- **Claude Sonnet 4.5 Thinking** - Excelente para coding
- **Gemini 3 Pro** - Modelo novo do Google
- **Gemini 3 Flash** - Rápido e gratuito

## Passo 1: Instalar o Antigravity (se ainda não tiver)

1. Vai no navegador e digita: **antigravity.google**
2. Faz download pra Windows
3. Instala e faz login com tua conta Google

## Passo 2: Configurar no OpenCode

Já configurei o plugin `opencode-google-antigravity-auth` no teu setup!

## Passo 3: Autenticar

Abre o terminal e roda:

```bash
opencode auth login
```

Escolhe:
- **Google** como provider
- **OAuth with Google (Antigravity)**

Faz login com a mesma conta Google que usa no Antigravity.

## Passo 4: Usar os modelos

Depois de logado, no OpenCode você pode trocar de modelo:

```
opencode
# Digite: /models
```

E verás:
- `google/antigravity-claude-opus-4-5-thinking`
- `google/antigravity-claude-sonnet-4-5-thinking`
- `google/antigravity-gemini-3-pro`
- `google/antigravity-gemini-3-flash`

## ATENÇÃO ⚠️

- Os modelos do Antigravity são gratuitos pelo plano individual
- O plano gratuito tem limites semanais
- Se tu tiver Google One AI Pro (pago), os limites são maiores

## Quando usar cada um?

- **Gemini 2.5 Flash** → Tarefas rápidas, debugging
- **Claude Sonnet 4.5** → Coding geral, refactoring
- **Claude Opus 4.5** → Arquitetura, projetos complexos
- **Gemini 3 Pro** → Criar apps rapidão (vibe coding)
